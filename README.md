# Certificate Decision Assistant

## Project Overview
This project aims to provide a Certificate Decision Assistant that helps users in making informed decisions about certificate requests related to various services.

## Architecture
The application is structured into a backend and a frontend section. The backend is powered by Flask and SQLAlchemy, while the frontend is built using React.

## Setup Instructions
1. **Clone the repository**: `git clone https://github.com/DebasishTripathy13/CA`
2. **Set up the backend**: Navigate to the backend directory and install dependencies from `requirements.txt`.
3. **Set up the frontend**: Navigate to the frontend directory and run `npm install` to install React dependencies.

## Technology Stack
- **Backend**: Python, Flask, SQLAlchemy
- **Frontend**: JavaScript, React
- **Database**: PostgreSQL
- **Cloud Services**: AWS (S3, etc.)
